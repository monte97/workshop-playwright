# Playwright Demo Application

This is a simple web application created for demonstrating Playwright testing techniques during the workshop.

## Features

- Registration form with validation
- Dynamic content loading
- Counter with increment/decrement/reset
- Hidden content that appears on button click
- Responsive design

## How to run

1. Navigate to the `demo-app` directory
2. Start a local server (e.g., using Python: `python -m http.server 8000`)
3. Open your browser to `http://localhost:8000`

## Purpose

This demo app provides various elements and interactions that can be used to practice writing Playwright tests, including:
- Form filling and submission
- Button clicks
- Working with dynamic content
- Handling hidden elements
- Working with different input types
